# Mi CRM Basic — NexCRM

Plataforma CRM todo en uno: gestión de clientes, e-commerce digital y captación de leads en redes sociales.

## Despliegue en Railway

Este proyecto es un sitio estático HTML. Para desplegarlo en Railway:

1. Conecta este repositorio en Railway
2. Railway detectará automáticamente el `index.html`
3. Selecciona el servicio **Static Site** o usa Nixpacks

## Stack

- HTML5 / CSS3 puro (sin dependencias)
- Diseño minimalista oscuro 2026
- Totalmente responsive
